Instrucciones de Test

  • Desktop: El input debe estar centrado y el botón a la derecha
  • Tablet (768px): El formulario debe ser más compacto pero mantener el layout horizontal
  • Móvil (480px): El formulario debe ser compacto con el input centrado y botón a la derecha
  • Diseño: Debe ser minimalista con sombras sutiles y bordes redondeados

Características del Nuevo Diseño:

  • ✅ Input centrado y expandible (flex: 1)
  • ✅ Botón a la derecha del input (flex-shrink: 0)
  • ✅ Diseño minimalista con sombras sutiles
  • ✅ Responsive para todos los dispositivos
  • ✅ Input y botón alineados horizontalmente
  • ✅ Espaciado consistente entre elementos
